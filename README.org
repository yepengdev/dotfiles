#+TITLE: dotfiles
#+SUBTITLE: managed with chezmoi — Gentoo Linux / niri Wayland

* 快速开始（新机器）

** 安装 chezmoi

#+begin_src sh
# Gentoo
sudo emerge --ask app-admin/chezmoi

# Other distros
# curl -sfL https://git.io/chezmoi | sh
#+end_src

** 克隆并应用

#+begin_src sh
chezmoi init --apply git@github.com:yepengdev/dotfiles.git
#+end_src

这会：
1. Clone 仓库到 ~/.local/share/chezmoi/
2. 将所有 dotfiles 安装到 home 目录
3. 自动跳过 .git 目录和插件/自动生成文件

** ⚠ 部署后需要手动操作

新机器上执行完 ~chezmoi init --apply~ 后：

#+begin_src sh
# 1. 部署 portage 系统配置（需要 root）
chezmoi apply   # 安装 run_once_deploy-portage.sh
              # 脚本会自动复制 ~/.config/portage/ → /etc/portage/

# 2. 恢复已安装的软件包
sudo emerge --ask $(cat ~/.config/portage-world)

# 3. 安装 niri_tweaks 依赖
# Python3 和 jq 通常已预装，检查：
which python3 jq
#+end_src

* 日常使用

| 命令                        | 作用                        |
|-----------------------------+-----------------------------|
| ~chezmoi diff~               | 查看本地与源的差异          |
| ~chezmoi apply~              | 应用变更到 home 目录        |
| ~chezmoi re-add ~/path~      | 将文件变更同步回 chezmoi    |
| ~chezmoi add ~/path~         | 将新文件纳入 chezmoi 管理   |
| ~chezmoi status~             | 查看管理状态                |
| ~chezmoi update~             | 从远程拉取最新配置          |
| ~chezmoi cd~                 | 跳到 source 目录            |

** 典型工作流

#+begin_src sh
# 修改某个配置后：
chezmoi re-add ~/.config/niri/cfg/keybinds.kdl

cd ~/.local/share/chezmoi
git commit -m "niri: ..."
git push
#+end_src

* 文件命名规则

chezmoi 在 source 目录使用前缀映射路径：

| 前缀                              | 映射到                    | 示例                                                                 |
|-----------------------------------+---------------------------+----------------------------------------------------------------------|
| ~dot_~                            | ~.~ 开头                   | ~dot_bashrc~ → ~/.bashrc~                                            |
| ~dot_config/~                     | ~/.config/~ 下            | ~dot_config/niri/config.kdl~ → ~/.config/niri/config.kdl~            |
| ~executable_dot_~                 | 可执行 + .开头             | ~executable_dot_xinitrc~ → ~/.xinitrc~（可执行）                     |
| ~private_~                        | 敏感文件（权限 600）      | ~private_fcitx5/~ → ~/.config/fcitx5/~（含密钥/令牌）                |
| ~symlink_~                        | 符号链接                   | ~symlink_pipewire.service~ → 软链到目标位置                          |
| ~run_once_~                       | 仅在首次 ~apply~ 执行一次  | ~run_once_deploy-portage.sh.tmpl~ → 新机自动运行                     |
| ~dot_config/portage/~ → /etc/  | 见下方说明                | ~dot_config/portage/make.conf~ → 通过 deploy 脚本复制到 /etc/portage/ |

~make.conf~ 等系统文件不在 home 目录下，chezmoi 无法直接管理。
做法：在 ~.config/portage/~ 维护副本，用 ~run_once~ 脚本部署到 /etc/portage/。

* 新机器完整恢复流程

将在新 Gentoo 机器上恢复到与当前一致的状态：

#+begin_src sh
# 1. 基础系统
# 确保已安装：Git, chezmoi, sudo doas

# 2. 克隆 dotfiles
chezmoi init --apply git@github.com:yepengdev/dotfiles.git

# 3. 部署系统配置
sudo ~/.local/share/chezmoi/run_once_deploy-portage.sh.tmpl
# 或 chezmoi apply（自动触发 run_once 脚本）

# 4. 安装所有软件包（234 个）
sudo emerge --ask $(cat ~/.config/portage-world)

# 5. ✅ 重启即可使用 niri
#+end_src

* 注意事项

| 注意点                      | 说明                                                    |
|-----------------------------+---------------------------------------------------------|
| ~.git~ 目录冲突             | fish、niri、doom 自带 .git，chezmoi 通过 ~.chezmoiignore~ 排除 |
| 插件文件                    | Fisher 插件、tide、nvim 插件等均通过 ignore 排除         |
| 含密钥的目录                | fcitx5（private_）、noctalia（未管理）含 API key/令牌     |
| niri_tweaks 依赖            | 需要 python3 和 jq                                      |
| 配置重载                    | niri 自动监视文件变更，保存即生效                        |
| Chezmoi 同步                | 修改后记得 ~chezmoi re-add~ + git commit + push          |
